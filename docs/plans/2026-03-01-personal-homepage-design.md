# 个人主页设计文档

**日期**: 2026-03-01
**类型**: GitHub Pages 静态个人主页

## 1. 项目概述

- **目标**: 为学生打造一个炫酷的个人主页
- **技术栈**: 纯 HTML + CSS + JavaScript
- **托管**: GitHub Pages

## 2. 页面结构

| 区域 | 内容 |
|------|------|
| Hero 区 | 姓名 + 标题 + 粒子动画背景 |
| 关于我 | 简介卡片（学校、身份、兴趣） |
| 技能 | 标签云展示技术栈 |
| 项目作品 | 3-4 个项目卡片 |
| 博客 | 文章列表卡片 |
| 联系 | GitHub/邮箱/社交链接 |

## 3. 视觉设计

### 配色方案
- 背景: 深色 (#0a0a0f)
- 主色: 渐变紫蓝 (#667eea → #764ba2)
- 强调色: 青色 (#00d9ff)
- 文字: 白色 (#ffffff) / 灰色 (#a0a0a0)

### 风格
- 科技感 / 赛博朋克
- 玻璃拟态卡片
- 动态粒子背景

## 4. 特效

- 🎯 particles.js 粒子背景（鼠标交互）
- ✨ 卡片 hover 发光/浮起效果
- ⭐ 页面加载入场动画 (fade-in, slide-up)
- 🌟 技能标签微动效

## 5. 响应式

- 桌面端 (>1024px): 多列布局
- 平板端 (768-1024px): 双列
- 手机端 (<768px): 单列堆叠
